This walkthrough demo is intended for vendors or anyone who wants to receive payments through crypto.

Each stage takes the vendor through the one-time configuration inputs needed to generate a payment.

In the end you can see the payment UI from the client's perspective:

  • No need to connect a wallet
  • No need to wait for the payment
  • Instant and secure

Select Payment Chains

Select the chains through which you want to be able to receive payments and, in case you want the payment to be immediately redirected to your wallet, enter it here.

Set Payment Amount
